# Off-Road-Autonomous-Vehicles-Semantic-Segmentation-and-Spatial-Overlay-Video-Assembly
(1) a large-scale off-road dataset designed for unstructured environments, (2) a Confusion-Aware Loss (CAL) that mitigates systematic misclassification in semantic segmentation, and (3) a spatial overlay video encoding scheme that improves compression efficiency by preserving semantic regions of interest.
